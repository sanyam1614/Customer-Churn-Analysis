Key findings

The numbers that matter

42.7%

Month-to-month contract churn — 15× higher than two-year contracts at 2.8%. Contract type is the single strongest churn predictor.

47–52%

Churn rate in the 0–12 month tenure window. New customers are the most vulnerable segment by far.

<6%

Churn rate for customers who pass 36 months of tenure. Loyalty compounds — reaching this milestone is a strong retention signal.

$996K

Combined monthly revenue at risk across the top 3 highest-churn segments. Retaining even half has outsized P&L impact.

Fiber optic

Customers on fiber optic internet churn at a significantly higher rate than DSL or no-internet subscribers, despite (or because of) higher monthly charges — pricing sensitivity and service quality are likely compounding factors.

Quickstart

Run it in under two minutes

# 1. Clone the repo
git clone https://github.com/sanyam1614/Customer-Churn-Analysis
cd Customer-Churn-Analysis

# 2. Install dependencies
pip install -r requirements.txt

# 3. Download dataset from Kaggle (free, no login needed via API)
#    https://www.kaggle.com/datasets/blastchar/telco-customer-churn

# 4. Run the analysis
python churn_analysis.py --input WA_Fn-UseC_-Telco-Customer-Churn.csv

# Outputs produced:
  churn_eda_report.png   ← 7-chart EDA dashboard
  CHURN_REPORT.md        ← executive summary report
